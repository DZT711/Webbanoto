## ğŸ’¾ Database Setup

### Database Configuration

This project uses the MySQL database named **`webbanoto`**

### Installation Steps

1. **Create the database:**
   ```sql
   CREATE DATABASE webbanoto;
   ```

2. **Import the SQL file:**
   - Locate the SQL file in the `Web_SQL` directory
   - Import it using phpMyAdmin or command line:
   ```bash
   mysql -u your_username -p webbanoto < Web_SQL/webbanoto.sql
   ```

3. **Configure database connection:**
   - Update your PHP configuration files with the correct database credentials
   - Ensure your local server is running on `http://php.local/`

## ğŸš€ Getting Started

### Prerequisites

- PHP 7.4 or higher
- MySQL 5.7 or higher
- Apache/Nginx web server
- Modern web browser

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/DZT711/Webbanoto.git
   ```

2. **Set up your local environment:**
   - Configure your web server to point to the project directory
   - Ensure the URL matches: `http://php.local/WebbanotoPHP/`

3. **Import the database** (see Database Setup section above)

4. **Access the application:**
   - Admin Portal: [http://php.local/WebbanotoPHP/admin/login](http://php.local/WebbanotoPHP/admin/login)
   - User Portal: [http://php.local/WebbanotoPHP/User](http://php.local/WebbanotoPHP/User)
